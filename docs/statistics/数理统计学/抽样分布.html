抽样分布

抽样分布是指从总体中抽取随机样本，得到的样本统计量的分布。样本统计量可以是样本平均数、标准差、比例等等。抽样分布是统计学中重要的概念，因为它与统计推断密切相关。

当我们收集数据时，我们可能无法完全检查或分析总体的全部数据。因此，我们从总体中抽取了一个样本来代表总体。抽样分布将描述从总体中所有可能的样本组合中，某个样本统计量的可能值的分布。这样就可以通过对抽样分布进行分析并在这个分布中计算概率来推断总体参数。

接下来，我们使用 Python 来说明抽样分布是如何计算的。首先，我们创建一个随机的数据集，表示总体：

import numpy as np

# 创建一个均值为 20、标准差为 5 的正态分布的总体
population = np.random.normal(20, 5, size=1000)

现在我们有了总体，我们可以使用 numpy 库中的 random.choice() 方法从总体中随机抽取一些样本：

# 从总体中随机抽取一个大小为 50 的样本
sample = np.random.choice(population, 50)

现在我们有了一个样本，让我们假设我们想计算样本平均值的抽样分布。为此，我们将抽取 1000 个随机样本，计算每个样本的平均值，并将这些平均值存储到另一个数组中。

# 计算 1000 个样本的平均值
mean_distribution = np.zeros(1000)
for i in range(1000):
    sample = np.random.choice(population, 50)
    mean_distribution[i] = np.mean(sample)

现在我们已经计算了 1000 个样本均值，我们可以使用 matplotlib 库将这个分布可视化出来。

import matplotlib.pyplot as plt

# 绘制抽样分布的直方图
plt.hist(mean_distribution, bins=50)
plt.xlabel('Sample Mean')
plt.ylabel('Frequency')
plt.title('Sampling Distribution of the Sample Mean')
plt.show()

下图显示了我们计算的抽样平均数的分布：

通过计算抽样分布，我们可以了解样本统计量的变异性。例如，在上面的例子中，我们看到样本平均值在总体均值附近变化很小，而在总体标准差较大时变化较大。这种变异性对于我们在节约时间和成本方面权衡样本大小时具有重要意义。
