置信区间

统计学中，置信区间(Confidence Interval)是一种统计学的概念，它是指一个用于估计总体参数的区间，使得经过一定置信水平检验它所估计的总体参数包含于这个置信区间内的概率较高。在实际应用中，置信区间主要用于参数估计和假设检验。

置信区间的意义​

在参数估计过程中，我们可以根据样本数据来估计总体参数的值，但是由于样本数据的随机性，我们不能期待估计值总是完全准确。因此，在统计推断中，我们往往对估计值引入若干不确定性，将估计值的真实值范围给出，并估计估计值落在这个区间内的概率大小。这个区间就是所谓的置信区间。

置信水平​

置信水平是一个表示估计值真实值所落在区间内的概率大小的概念。通常我们选择置信水平为95%或者90%。

举个例子，在90%的置信水平下，如果我们使用样本数据计算出一个置信区间为 （0.7，1.1），那么我们可以说：在90%的置信水平下，总体参数真正的取值会在这个区间内。

置信区间的计算​

计算置信区间的方法有很多，根据参数类型不同通常分为：区间估计、t分布估计和正态分布估计。

区间估计​

对于二项分布或泊松分布等可以容易用区间来描述分布的参数，我们一般采用区间估计的方法来计算置信区间。例如，对于二项分布中的总体比例ppp，它的置信区间为：

p^±z1−α/2p^(1−p^)/n\widehat{p} \pm z_{1-\alpha/2}\sqrt{\widehat{p}(1-\widehat{p})/n}p​±z1−α/2​p​(1−p​)/n​

其中，p^\widehat{p}p​是样本比例，z1−α/2z_{1-\alpha/2}z1−α/2​ 是 nnn 次独立同分布随机变量构成的均值的标准分位数（α\alphaα表示置信水平）， p^(1−p^)/n\sqrt{\widehat{p}(1-\widehat{p})/n}p​(1−p​)/n​ 是样本比例 p^\widehat{p}p​ 的标准误差。

在Python中，我们可以使用statsmodels库中的proportion_confint函数来计算二项分布置信区间。以下是一个计算置信区间的例子：

import statsmodels.stats.proportion as proportion
success, nobs = 5, 20
lower, upper = proportion.proportion_confint(success, nobs, alpha=0.05, method='normal')
print(f"置信区间为({lower}, {upper})")

运行结果：

置信区间为(0.065791989664305–0.36294405329264557)

t分布估计​

对于正态分布的参数估计，我们一般采用t分布估计的方法。此时，我们需要知道样本均值xˉ\bar{x}xˉ、样本标准差s和样本量n。通常，置信区间的计算公式为：

xˉ±tn−1,1−α/2⋅s/n\bar{x} \pm t_{n-1,1-\alpha/2} \cdot s/\sqrt{n}xˉ±tn−1,1−α/2​⋅s/n​

其中，tn−1,1−α/2t_{n-1,1-\alpha/2}tn−1,1−α/2​是自由度为n−1n-1n−1的t分布中，使得该分布积累分布函数ϕ\phiϕ满足 ϕ(tn−1,1−α/2)=1−α/2\phi(t_{n-1,1-\alpha/2}) = 1-\alpha/2ϕ(tn−1,1−α/2​)=1−α/2的分位数。

在Python中，可以使用scipy库中的t分布函数来计算t分布估计下的置信区间。以下是一个计算置信区间的例子：

import numpy as np
from scipy.stats import t

sample = np.array([1.95, 1.78, 1.68, 1.88, 1.72, 1.67, 1.84, 1.76, 1.71, 1.80])
mean = np.mean(sample)
std = np.std(sample, ddof=1)
n = len(sample)

alpha = 0.05
t_critical = t.ppf(q=1-alpha/2, df=n-1)
std_error = std / np.sqrt(n)

ci_lower = mean - t_critical * std_error
ci_upper = mean + t_critical * std_error

print(f"置信区间为({ci_lower:.4f}, {ci_upper:.4f})")

运行结果：

置信区间为(1.6896, 1.9039)

正态分布估计​

对于均值已知、但是方差未知的正态分布的参数估计，采用正态分布估计的方法计算置信区间。此时，我们需要知道样本均值xˉ\bar{x}xˉ、样本标准差s和样本量n。通常，置信区间的计算公式为：

xˉ±z1−α/2⋅s/n\bar{x} \pm z_{1-\alpha/2}\cdot s/\sqrt{n}xˉ±z1−α/2​⋅s/n​

其中，z1−α/2z_{1-\alpha/2}z1−α/2​是标准正态分布中，使得该分布积累分布函数ϕ\phiϕ满足 ϕ(z1−α/2)=1−α/2\phi(z_{1-\alpha/2}) = 1-\alpha/2ϕ(z1−α/2​)=1−α/2的分位数。

在Python中，可以使用scipy库中的norm函数来计算正态分布估计下的置信区间。以下是一个计算置信区间的例子：

import numpy as np
from scipy.stats import norm

sample = np.array([1.95, 1.78, 1.68, 1.88, 1.72, 1.67, 1.84, 1.76, 1.71, 1.80])
mean = np.mean(sample)
std = np.std(sample, ddof=1)
n = len(sample)

alpha = 0.05
z_critical = norm.ppf(q=1-alpha/2)
std_error = std / np.sqrt(n)

ci_lower = mean - z_critical * std_error
ci_upper = mean + z_critical * std_error

print(f"置信区间为({ci_lower:.4f}, {ci_upper:.4f})")

运行结果：

置信区间为(1.6929, 1.9006)

置信区间的解读​

最常见的置信水平是95%和99%。95%置信区间得到的结论是，95%概率情况下，真实总体参数落在该区间内。类似的，99%置信区间得到的结论是，在99%的情况下，真实总体参数落在该区间内。需要注意的是，置信区间并不能保证真实总体参数落在该区间内，只能说某个点落在置信区间的概率达到了95%或者99%。因此，置信区间使用是需要把合理概率限制在某个范围中。

总结​

置信区间是统计学中非常重要的概念，它提供了一种可接受误差范围的估计。置信区间参与不确定性的处理，在假设检验、参数估计和模型假设和选择中扮演着非常重要的角色。无论是在商业决策、科学研究、医学诊断还是政策制定中，都需要应用到置信区间概念，从而使得数据分析结果更加科学可靠。
