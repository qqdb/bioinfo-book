样本

在统计学中，样本（sample）是指从总体中随机选取的一部分个体，用来作出总体的估计或推断。可以理解为，样本是一个总体的代表。

样本的选取方法​

  1. 简单随机抽样（simple random sampling）

简单随机抽样是指从总体中随机抽取n个个体，每个个体被抽中的概率相等。简单随机抽样非常常用，是其他抽样方法的基础。

在Python中，可以用random模块来进行抽样。比如从1~100中随机抽取10个数：

import random

sample = random.sample(range(1, 101), 10)
print(sample)
  1. 分层抽样（stratified sampling）

分层抽样是指将总体分成若干个层，然后从各个层中分别抽出样本。这种方法可以针对总体的特点进行更准确的估计。

举个例子，假设我想要对一所学校的学生的成绩进行调查，但是学校有不同年级和不同专业的学生，如果我采用简单随机抽样，可能会导致各年级各专业的学生被抽到的数量不均衡。这时，我可以对不同年级和不同专业的学生分别采用简单随机抽样，以保证样本更具代表性。

在Python中，可以先将总体根据某些特征进行分类，然后再对每个分类进行抽样。比如对1~100抽样，要求偶数和奇数的个数相等：

even_nums = [num for num in range(1, 101) if num % 2 == 0] # 偶数
odd_nums = [num for num in range(1, 101) if num % 2 == 1] # 奇数

even_sample = random.sample(even_nums, 25) # 从偶数中随机抽取25个数
odd_sample = random.sample(odd_nums, 25) # 从奇数中随机抽取25个数

sample = even_sample + odd_sample # 将两个样本合并
print(sample)
  1. 系统抽样（systematic sampling）

系统抽样是指按照一定规则选取个体，比如从总体中每隔k个个体选取一个。如果总体的个体是有序的，那么系统抽样是非常方便的。

比如，我想要从1~100中每隔5个数选取一个数：

k = 5
sample = [num for num in range(1, 101, k)]
print(sample)

样本的应用​

样本是统计学中非常重要的概念，通过对样本的分析可以得出总体的参数估计和推断。

比如，我想要估计全国人口的平均年龄，但是不可能调查全国每个人的年龄，因此我可以采用抽样的方法，随机选取一部分人口的年龄进行调查。假设我抽取了1000个人口，他们的平均年龄是27岁，那么我可以将这个样本均值作为全国人口平均年龄的估计值。

在Python中，可以通过numpy模块来计算样本的均值、方差等统计量：

import numpy as np

sample = np.random.normal(loc=0, scale=1, size=1000) # 生成1000个标准正态分布的数

mean = np.mean(sample) # 计算样本均值
var = np.var(sample) # 计算样本方差

print("样本均值：{}".format(mean))
print("样本方差：{}".format(var))

总结​

样本是统计学中非常重要的概念，通过对样本的分析可以得出总体的参数估计和推断。我们可以使用不同的抽样方法来选取样本，以保证样本的代表性和准确性。在Python中，我们可以使用random、numpy等模块来进行样本的生成和统计计算。
