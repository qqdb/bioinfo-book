马尔科夫链蒙特卡罗 (MCMC)方法

概述​

马尔科夫链蒙特卡罗 (Markov Chain Monte Carlo (MCMC) Methods)方法是一种用于模拟复杂概率分布的方法。它可以用于模拟一系列随机变量，而这些随机变量的分布往往非常复杂，难以直接计算。

MCMC 方法的基本思想是利用马尔科夫链的性质，生成一系列随机样本，并通过这些样本来估计目标分布的性质。MCMC 方法的核心在于如何构建一个合适的马尔科夫链，使得该链的平稳分布即为目标分布。

马尔科夫链​

马尔科夫链是一种描述随机过程的数学模型。它由一系列状态和状态之间的转移概率组成，即 P(Xt=j∣Xt−1=i)P(X_t=j|X_{t-1}=i)P(Xt​=j∣Xt−1​=i) 表示在时刻 ttt，如果当前状态为 iii，则转移到状态 jjj 的概率。其中，XtX_tXt​ 表示随机变量在时刻 ttt 的状态。

如果一个随机变量 {Xt}\{X_t\}{Xt​} 满足马尔科夫性质，则有：

P(Xt=j∣Xt−1=i,X0=j0,…,Xt−2=jt−2)=P(Xt=j∣Xt−1=i)P(X_t=j|X_{t-1}=i,X_0=j_0,\ldots,X_{t-2}=j_{t-2})=P(X_t=j|X_{t-1}=i)P(Xt​=j∣Xt−1​=i,X0​=j0​,…,Xt−2​=jt−2​)=P(Xt​=j∣Xt−1​=i)

即在已知过去的所有状态的条件下，当前状态的概率只与前一时刻的状态有关，而与其它时刻的状态无关。

蒙特卡罗方法​

蒙特卡罗方法是一种利用随机数估计复杂问题的数值方法。它通过随机抽样来获取样本，再根据样本进行估计。

例如，要估计一个函数 f(x)f(x)f(x) 在区间 [a,b][a,b][a,b] 上的积分，可以通过随机抽样来获得样本点 {xi}∼U(a,b)\{x_i\}\sim U(a,b){xi​}∼U(a,b)，然后利用蒙特卡罗积分公式：

∫abf(x) dx≈1N∑i=1Nf(xi)(b−a)\int_a^b f(x)\,\mathrm{d}x\approx\frac{1}{N}\sum_{i=1}^N f(x_i)(b-a)∫ab​f(x)dx≈N1​∑i=1N​f(xi​)(b−a)

来估计积分值。其中，NNN 为样本数。

马尔科夫链蒙特卡罗 (MCMC) 方法​

马尔科夫链蒙特卡罗 (MCMC) 方法是结合了马尔科夫链和蒙特卡罗方法的一种复杂概率分布模拟方法。它通过构造一个马尔科夫链来模拟目标分布，然后利用蒙特卡罗积分公式来估计目标分布的性质。

MCMC 方法的核心在于如何构造一个合适的马尔科夫链。如果构造的马尔科夫链满足以下两个条件，则称其为可逆马尔科夫链：

  1. 细致平衡条件：对于任意状态 i,ji,ji,j，有 P(i,j)f(i)=P(j,i)f(j)P(i,j)f(i)=P(j,i)f(j)P(i,j)f(i)=P(j,i)f(j)，其中 f(i)f(i)f(i) 为目标分布在状态 iii 处的概率密度函数。
  2. 不可约条件：任意两个状态之间都可以相互转换。

如果可逆马尔科夫链的平稳分布即为目标分布，则称该链为马尔科夫链蒙特卡罗 (MCMC) 链。

Python 实现​

以下是一个简单的 Python 实现示例。假设要模拟目标分布为 f(x)=12exp⁡(−∣x∣)f(x)=\frac{1}{2}\exp(-|x|)f(x)=21​exp(−∣x∣) 的随机变量，可以使用 Metropolis-Hastings 算法构造一个可逆马尔科夫链。

import numpy as np

def target(x):
    """
    目标分布
    """
    return 0.5*np.exp(-np.abs(x))

def proposal(x):
    """
    生成候选值
    """
    return np.random.normal(x, 1.0)

def acceptance(x, y):
    """
    接受率
    """
    alpha = target(y)/target(x)*np.exp(-0.5*(y-x)**2)
    return min(alpha, 1.0)

def mcmc(num_samples):
    """
    MCMC 模拟
    """
    samples = np.zeros(num_samples)
    x = 0.0
    for i in range(num_samples):
        y = proposal(x)
        alpha = acceptance(x, y)
        if np.random.uniform() < alpha:
            x = y
        samples[i] = x
    return samples

在上面的实现中，使用了 Metropolis-Hastings 算法构造了一个可逆马尔科夫链，其中目标分布为 f(x)f(x)f(x)，生成的候选值为 N(x,1)\mathcal{N}(x,1)N(x,1) 的正态分布，接受率为 α=min⁡{1,f(y)f(x)p(x∣y)p(y∣x)}N(y∣x,1)N(x∣y,1)\alpha=\min\left\{1,\frac{f(y)}{f(x)}\frac{p(x|y)}{p(y|x)}\right\}\frac{\mathcal{N}(y|x,1)}{\mathcal{N}(x|y,1)}α=min{1,f(x)f(y)​p(y∣x)p(x∣y)​}N(x∣y,1)N(y∣x,1)​，其中 p(x∣y)p(x|y)p(x∣y) 和 p(y∣x)p(y|x)p(y∣x) 均为正态分布的概率密度函数。

最后，可以使用上述函数生成 MCMC 链，并调用其他统计学工具进行分析。例如，可以使用 Matplotlib 绘制该链的直方图和核密度估计图，以及计算该链的均值和标准差等统计量。

import matplotlib.pyplot as plt

samples = mcmc(10000)
print("Mean:", np.mean(samples))
print("Std. Dev.:", np.std(samples))

fig, axes = plt.subplots(nrows=2, ncols=1)
axes[0].hist(samples, bins=50, density=True, alpha=0.5)
x = np.linspace(-5, 5, 100)
axes[0].plot(x, target(x), 'r-')
axes[0].set_xlabel("x")
axes[0].set_ylabel("Density")
axes[0].legend(["Target", "Samples"])

axes[1].plot(samples)
axes[1].set_xlabel("Iteration")
axes[1].set_ylabel("Sample Value")

plt.show()

在上述实现中，使用了 Matplotlib 绘制了 MCMC 链的直方图和核密度估计图，以及绘制了该链的随机样本图。

综上所述，马尔科夫链蒙特卡罗 (MCMC) 方法是一种重要的概率模拟方法，并已广泛应用于统计学、贝叶斯推断、机器学习等领域。
