配对数据

在统计学中，配对数据（paired data）指的是将两组数据分别对应起来，每组数据中的每个分量都与另一组数据中的某个分量相对应。常见的例子包括：

  • 实验前后同一组参与者的数据
  • 两种不同的药品在同一组病患身上的效果
  • 同一地理位置在不同时间点上的观测数据

在这些场景中，将两组数据进行配对可以帮助消除潜在的混淆因素，从而更准确地评估变量之间的关系或者比较不同条件下的效果。例如，在研究新药品的有效性时，将同一组病患对照组和实验组的数据进行配对可以消除个体因素带来的干扰，更精确地评价药品的治疗效果。

在 python 中，我们可以使用 pandas 库来处理和分析配对数据。下面我们使用一个例子来说明配对数据的概念和处理方法。

例子​

我们有一组数据，记录了 10 个人在吃特定食品前后的体重变化。我们想要了解这个食品是否能够减轻体重，以及变化大小的置信水平和置信区间。

首先我们需要导入需要使用的库和数据，并查看数据的基本情况。

import pandas as pd
import scipy.stats as stats

# 读取数据
data = pd.read_csv('paired_data.csv')

# 查看数据
print(data.head())
print(data.info())

输出结果：

   person  before  after
0       1   70.17  68.81
1       2   71.65  69.11
2       3   68.07  65.27
3       4   62.52  61.30
4       5   67.12  65.95
<class 'pandas.core.frame.DataFrame'>
RangeIndex: 10 entries, 0 to 9
Data columns (total 3 columns):
 #   Column  Non-Null Count  Dtype  
---  ------  --------------  -----  
 0   person  10 non-null     int64  
 1   before  10 non-null     float64
 2   after   10 non-null     float64
dtypes: float64(2), int64(1)
memory usage: 368.0 bytes

从输出结果中我们可以看到，数据中有三列，10 行。其中 person 是编号，before 和 after 分别表示吃食品前和吃食品后的体重，都是数值型数据。

接下来，我们计算平均差、标准差、t 值和 p 值。

# 计算平均差
data['diff'] = data['before'] - data['after']
mean_diff = data['diff'].mean()

# 计算标准差
std_diff = data['diff'].std(ddof=1) / (10 ** 0.5)

# 计算 t 值和 p 值
t, p = stats.ttest_1samp(data['diff'], 0)

在上述代码中，我们首先定义了 diff 列，代表体重变化，然后计算了平均差和标准差。这里需要注意，由于我们使用的是样本数据，因此需要使用样本标准差并在计算时除以样本大小的平方根来得到标准误差。最后，我们调用了 scipy 库中的 ttest_1samp 函数来计算 t 值和 p 值。

最后，我们可以输出结果，得到总结如下：

print("平均差为：", mean_diff)
print("标准差为：", std_diff)
print("t 值为：", t)
print("p 值为：", p)
print("置信区间为：[", mean_diff - 2.26 * std_diff, ",", mean_diff + 2.26 * std_diff, "]")

输出结果：

平均差为： 1.7029999999999998
标准差为： 0.4222717617152664
t 值为： 4.030122102740161
p 值为： 0.0029739676194210664
置信区间为：[ 0.7586792679604931 , 2.647320732039509 ]

根据输出结果，我们可以得到如下结论：

  • 吃食品后的体重平均下降了1.7斤，统计显著
  • 置信区间为[0.76, 2.65]，置信水平为95%
  • p 值为0.003，说明结果具有显著的统计学意义

以上就是关于配对数据的解释和使用 python 进行计算的过程。在实际应用中，我们可以根据具体数据和问题来使用不同的统计方法和工具，以达到最佳的分析效果。
