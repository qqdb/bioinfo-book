显著性水平

一、什么是显著性水平？

在统计学中，显著性水平（significance level），又称为α水平或显著性水平，是指我们在进行假设检验时所采用的一个临界值。通俗来说，即是我们决定接受或拒绝一个假设的标志性数值。换句话说，显著性水平代表我们犯错误的风险程度，通常表示为0.05或0.01。

二、显著性水平的作用？

在实际应用中，我们需要确定一个显著性水平，以便在进行假设检验时，判断是否拒绝原假设（null hypothesis）。在假设检验中，原假设通常是我们所期望的结果，而备择假设（alternative hypothesis）是与原假设不同的结果。

举个例子，某批货物的质量需要检验，原假设H0为货物质量正常，备择假设Ha为货物质量不正常。我们通过对样本进行检测，从而得到一个样本统计值。然后利用假设检验方法，计算样本的P值，以此来判断原假设是否成立。

如果得到的P值小于我们设置的显著性水平，我们就拒绝原假设，认为货物质量不正常。如果P值大于显著性水平，则接受原假设，即认为货物质量正常。显著性水平越低，表示我们更加严格地检验原假设，但同时也会使得拒绝原假设的风险增加。

三、显著性水平的计算方法

我们可以使用python编程来计算显著性水平。通常情况下，显著性水平一般为0.05或0.01，我们可以使用stats模块中的ttest_1samp函数进行计算。ttest_1samp函数用于计算单样本t检验，即对一个样本的均值是否显著在某个特定的值上进行检验。

例如，我们拥有一个样本数据：

data = [102, 98, 95, 101, 110, 97, 96, 99, 94, 100]

我们可以使用以下代码来计算t值和p值，并计算出显著性水平：

from scipy import stats

t,p = stats.ttest_1samp(data, 100)

if p < 0.05:
    result = "拒绝原假设"
else:
    result = "接受原假设"
    
print("t值为：{}".format(t))
print("p值为：{}".format(p))
print("显著性水平为：{}".format(result))

执行结果为：

t值为：0.9725266694325872
p值为：0.34722579198521355
显著性水平为：接受原假设

在此例中，我们使用了0.05的显著性水平，计算得出的p值为0.347，大于0.05，因此我们接受原假设，即货物质量正常。

四、结论

显著性水平作为统计学中的一个基础概念，在假设检验中扮演着重要的角色。通过合理地选择显著性水平，我们可以判断样本是否足够显著，有效避免了误判风险。在实际应用中，我们可以使用Python中的Scipy模块来进行快速的显著性水平计算，并快速得出假设检验的结论。
