多维随机变量和分布

在统计学中，随机变量通常是指一个数学函数，它将随机事件映射到实数值域上。多维随机变量则是指具有多个随机变量组成的向量。例如，当我们同时考虑一个人的身高和体重时，可以将其视为一个具有两个随机变量的向量。

对于一个多维随机变量而言，我们通常会关注该变量的分布情况，因为分布可以描述该变量在不同取值下的概率密度。一般地，多维随机变量的分布可以使用联合分布、条件分布及边缘分布等方式描述，下面来依次介绍。

联合分布​

对于多维随机变量 (X1,X2,...,Xn)(X_1, X_2, ..., X_n)(X1​,X2​,...,Xn​)，它们的联合分布可以用一个函数 fX1,X2,...,Xn(x1,x2,...,xn)f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n)fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​) 描述，它表示当所有随机变量均取某个特定值时，该事件出现的概率密度。该函数需要满足如下条件：

  1. fX1,X2,...,Xn(x1,x2,...,xn)≥0f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n) \geq 0fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​)≥0；
  2. ∫−∞∞...∫−∞∞fX1,X2,...,Xn(x1,x2,...,xn)dx1dx2...dxn=1\int_{-\infty}^{\infty}...\int_{-\infty}^{\infty} f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n)dx_1dx_2...dx_n = 1∫−∞∞​...∫−∞∞​fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​)dx1​dx2​...dxn​=1；
  3. 对于任意的可测集 A⊂RnA \subset \mathbb{R}^nA⊂Rn，都有 P((X1,X2,...,Xn)∈A)=∫AfX1,X2,...,Xn(x1,x2,...,xn)dx1dx2...dxnP((X_1, X_2, ..., X_n) \in A) = \int_A f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n)dx_1dx_2...dx_nP((X1​,X2​,...,Xn​)∈A)=∫A​fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​)dx1​dx2​...dxn​。

其中第一个条件保证了概率密度的非负性，第二个条件确保了概率密度的归一化，第三个条件则邀请了概率的基本性质。

对于联合分布而言，我们通常还可以计算出每个随机变量的边缘分布、两个随机变量的协方差以及任意 kkk 个随机变量之间的联合分布等。

边缘分布​

当我们在考虑一个多维随机变量的某个分量 XiX_iXi​ 时，我们可能会想要计算该随机变量在 XiX_iXi​ 上的分布情况。这就是所谓的边缘分布，它是将联合分布中其他变量较为“舍弃”后得到的。

边缘分布可以通过对联合分布积分来计算，即：

fXi(xi)=∫−∞∞...∫−∞∞fX1,X2,...,Xn(x1,x2,...,xn)dx1dx2...dxi−1dxi+1...dxnf_{X_i}(x_i) = \int_{-\infty}^{\infty}...\int_{-\infty}^{\infty} f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n)dx_1dx_2...dx_{i-1}dx_{i+1}...dx_nfXi​​(xi​)=∫−∞∞​...∫−∞∞​fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​)dx1​dx2​...dxi−1​dxi+1​...dxn​

需要注意的是，这里的积分是对所有 Xj(j≠i)X_j (j \neq i)Xj​(j=i) 进行积分。在计算过程中，我们可以使用 python 中的 scipy 库来实现。例如，对于二维正态分布 N(μ,Σ)N(\mu, \Sigma)N(μ,Σ)，其边缘分布可以如下计算：

import numpy as np
from scipy.stats import multivariate_normal

mu = np.array([0, 0])
cov = np.array([[1, 0.5], [0.5, 1]])

rv = multivariate_normal(mean=mu, cov=cov)

# 计算 X1 的边缘分布
rv_X1 = rv.pdf(x=[1, np.inf]) - rv.pdf(x=[-1, np.inf])
print(rv_X1)

条件分布​

当我们已知一个多维随机变量中某些分量的取值时，我们可以计算其余变量的条件分布。例如，在已知一个人的身高时，我们可以计算该人体重的条件分布。

对于多维随机变量 (X1,X2,...,Xn)(X_1, X_2, ..., X_n)(X1​,X2​,...,Xn​)，我们可以定义某个条件下另一个随机变量 XiX_iXi​ 的条件分布为：

fXi∣Xj=xj(xi)=fX1,X2,...,Xn(x1,x2,...,xn)fXj(xj)f_{X_i|X_j=x_j}(x_i)=\dfrac{f_{X_1, X_2, ..., X_n}(x_1, x_2, ..., x_n)}{f_{X_j}(x_j)}fXi​∣Xj​=xj​​(xi​)=fXj​​(xj​)fX1​,X2​,...,Xn​​(x1​,x2​,...,xn​)​

需要注意的是，这里我们假设 fXj(xj)≠0f_{X_j}(x_j) \neq 0fXj​​(xj​)=0，即 XjX_jXj​ 的取值不为零。

同样地，我们可以使用 python 中的 scipy 库来计算条件分布。例如，对于二维正态分布 N(μ,Σ)N(\mu, \Sigma)N(μ,Σ)，条件分布可以如下计算：

import numpy as np
from scipy.stats import multivariate_normal

mu = np.array([0, 0])
cov = np.array([[1, 0.5], [0.5, 1]])

rv = multivariate_normal(mean=mu, cov=cov)

# 计算在 X1 = 1 的条件下，X2 的条件分布
rv_X2_given_X1 = multivariate_normal(mean=mu + np.dot(np.dot(cov, np.linalg.inv(cov[0])), np.array([1, 0])),
                                      cov=cov[1:, 1:]).pdf(x=np.array([0, 1]))
print(rv_X2_given_X1)

总结​

在统计学中，多维随机变量和分布是非常重要的概念。联合分布、边缘分布和条件分布是常见的分布类型，使用 python 编程可以很方便地计算它们的概率密度和分布情况。通过熟练掌握这些概念，我们可以更好地理解和应用统计学的相关知识。
