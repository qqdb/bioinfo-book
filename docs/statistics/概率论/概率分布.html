概率分布

概率分布（Probability distribution）是概率论和数理统计学中的一个重要概念，用来描述一个随机变量取各个值的概率大小。概率分布是统计学和机器学习中必不可少的一部分。对于给定的随机变量，可以通过概率分布来描述它的取值范围和概率大小，从而帮助我们更好地理解数据并进行预测，在实际应用中具有广泛的应用价值。

离散概率分布​

在离散概率分布中，随机变量只能取有限或者可数个值。离散变量的概率分布可以通过概率质量函数（Probability Mass Function）来描述，通常表示为 P(X=i)P(X=i)P(X=i) 表示随机变量 XXX 取值为 iii 的概率大小，且满足 P(X=i)≥0P(X=i) \geq 0P(X=i)≥0 和 ∑iP(X=i)=1\sum\limits_{i} P(X=i) = 1i∑​P(X=i)=1。

常见的离散概率分布包括：伯努利分布、二项分布、泊松分布等。

伯努利分布​

伯努利分布是最简单的离散概率分布，描述了一次随机试验中成功和失败的概率。如果随机变量 XXX 只有两个取值 {0,1}\{0,1\}{0,1}，且当 X=1X=1X=1 时，表示成功的概率为 ppp，当 X=0X=0X=0 时，表示失败的概率为 1−p1-p1−p。则可以表示伯努利分布的概率质量函数为：

$$ P(X=i) = \begin{cases} 1-p & i=0, \ p & i=1. \end{cases} $$

在 python 中可以使用 scipy 库中的 bernoulli() 方法来生成伯努利分布的概率质量函数，示例代码如下：

from scipy.stats import bernoulli

# 成功的概率为 0.6
p = 0.6
# 生成伯努利分布的概率质量函数
b = bernoulli(p)
# 随机取样 10 次，查看结果
print(b.rvs(10))

输出结果如下：

[0 0 0 1 1 1 1 0 1 0]

二项分布​

二项分布描述了在 nnn 次独立的伯努利试验中，成功的次数 kkk 的概率分布。记随机变量 XXX 表示在 nnn 次伯努利试验中成功的次数，成功的概率为 ppp，则可以表示二项分布的概率质量函数为：

$$ P(X=k) = \binom{n}{k} p^{k} (1-p)^{n-k} $$

其中，(nk)\binom{n}{k}(kn​) 表示组合数，表示在 nnn 个不同元素中选择kkk 个元素的方案数。在 python 中可以使用 scipy 库中的 binom() 方法来生成二项分布的概率质量函数，示例代码如下：

from scipy.stats import binom

# 10 次伯努利试验中，成功的概率为 0.6
n, p = 10, 0.6
# 生成二项分布的概率质量函数
b = binom(n, p)
# 查看随机变量取 2 的概率
print(b.pmf(2))

输出结果如下：

0.12093235200000015

表示在 10 次伯努利试验中，成功的概率为 0.6，随机变量取 2 的概率为 0.1209。

泊松分布​

泊松分布描述了在一定时间或空间内，随机事件发生次数的概率分布。记随机变量 XXX 表示在单位时间或单位空间内随机事件发生的次数，λ\lambdaλ 表示单位时间或单位空间内随机事件的平均发生次数，那么泊松分布的概率质量函数可以表示为：

$$ P(X=k) = \frac{\lambda^{k} e^{-\lambda}}{k!} $$

在 python 中可以使用 scipy 库中的 poisson() 方法来生成泊松分布的概率质量函数，示例代码如下：

from scipy.stats import poisson

# 单位时间内随机事件的平均发生次数为 2
lambda_ = 2
# 生成泊松分布的概率质量函数
p = poisson(lambda_)
# 查看随机变量取 3 的概率
print(p.pmf(3))

输出结果如下：

0.18044704431548356

表示在单位时间内随机事件的平均发生次数为 2，随机变量取 3 的概率为 0.1804。

连续概率分布​

在连续概率分布中，随机变量可以连续取任何一个实数值。连续变量的概率分布可以通过概率密度函数（Probability Density Function）来描述，通常表示为 f(x)f(x)f(x)，表示随机变量 XXX 在某个取值区间 [a,b][a,b][a,b] 中的概率为 ∫abf(x)dx\int_{a}^{b} f(x)dx∫ab​f(x)dx，同时满足 f(x)≥0f(x) \geq 0f(x)≥0 和 ∫−∞+∞f(x)dx=1\int_{-\infty}^{+\infty}f(x)dx = 1∫−∞+∞​f(x)dx=1。

常见的离散概率分布包括：均匀分布、正态分布、指数分布等。

均匀分布​

均匀分布表示在一个区间 [a,b][a,b][a,b] 上，随机变量 XXX 取任意一个实数的概率都是相同的，可以用概率密度函数 f(x)=1b−af(x)=\frac{1}{b-a}f(x)=b−a1​ 来描述。

在 python 中可以使用 scipy 库中的 uniform() 方法来生成均匀分布的概率密度函数，示例代码如下：

from scipy.stats import uniform
import matplotlib.pyplot as plt
import numpy as np

# 在区间 [0,1] 上产生一组均匀分布的数据
data = uniform.rvs(size=1000, loc=0, scale=1)

# 绘制概率密度函数的图像
fig, ax = plt.subplots(1, 1)
x = np.linspace(0, 1, 1000)
pdf = uniform.pdf(x, loc=0, scale=1)
ax.plot(x, pdf, 'r-', lw=2, alpha=0.6, label='uniform pdf')
ax.hist(data, density=True, histtype='stepfilled', alpha=0.2)
ax.legend(loc='best', frameon=False)
plt.show()

正态分布​

正态分布是最常用的连续概率分布之一，又称为高斯分布，描述了自然界和社会现象中许多随机变量的分布情况。正态分布的概率密度函数是一个钟形曲线，由两个参数 μ\muμ （期望）和 σ\sigmaσ （标准差）来决定形态，表示为 f(x)=12πσe−(x−μ)22σ2f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}f(x)=2π​σ1​e−2σ2(x−μ)2​。

在 python 中可以使用 scipy 库中的 norm() 方法来生成正态分布的概率密度函数，示例代码如下：

from scipy.stats import norm
import matplotlib.pyplot as plt
import numpy as np

# 在标准正态分布上产生一组数据
data = norm.rvs(size=1000, loc=0, scale=1)

# 绘制概率密度函数的图像
fig, ax = plt.subplots(1, 1)
x = np.linspace(-4, 4, 1000)
pdf = norm.pdf(x, loc=0, scale=1)
ax.plot(x, pdf, 'r-', lw=2, alpha=0.6, label='normalpdf')
ax.hist(data, density=True, histtype='stepfilled', alpha=0.2)
ax.legend(loc='best', frameon=False)

指数分布​

指数分布描述了连续时间下随机事件的时间间隔，可以用概率密度函数 f(x)=λe−λxf(x)=\lambda e^{-\lambda x}f(x)=λe−λx 来描述，其中 λ\lambdaλ 表示速率参数，也称为参数 β\betaβ 的倒数。

在 python 中可以使用 scipy 库中的 expon() 方法来生成指数分布的概率密度函数，示例代码如下：

from scipy.stats import expon
import matplotlib.pyplot as plt
import numpy as np

# 产生一个指数分布的数据
data = expon.rvs(size=1000, loc=0, scale=1)

ax.hist(data, density=True, histtype='stepfilled', alpha=0.2)
