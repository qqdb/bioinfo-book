Genome Browser

访问链接：https://genome.ucsc.edu/

介绍​

Genome Browser是一个开源的在线基因组浏览器，它提供了许多物种的基因组序列数据，包括人类、小鼠、果蝇、酵母等多个模式生物。用户可以通过输入基因名、染色体坐标等信息查询不同物种的基因组序列、基因注释、启动子区域、剪切异构体差异等信息。Genome Browser还提供了可视化工具，方便用户基于浏览器中的基因组图谱进行数据分析和比较，支持多种涉及人类基因组的研究应用。

应用​

Genome Browser被广泛应用于许多生物信息学研究领域，其中包括：

  • 基础的基因组浏览和注释
  • 基因调控机制和信号通路的探究
  • 转录组学和差异表达分析
  • 基于基因组序列的进化分析
  • 疾病遗传学和个体化医疗研究
